# Attendance System

A simple web-based attendance management system built with Node.js and Express.

## Features

- Mark attendance for students
- View all attendance records
- RESTful API for attendance data

## Setup

```bash
npm install
npm start
```

The server runs at http://localhost:3000.

## Tech Stack

- Node.js, Express
- Vanilla JavaScript, HTML, CSS
- JSON file storage

## Project Structure

```
attendance-system/
├── server.js              # Express server
├── package.json
├── data/

│   └── public/
```
